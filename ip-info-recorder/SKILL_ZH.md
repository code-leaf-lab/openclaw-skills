# IP 信息记录技能

## 描述
自动化获取全面的公网 IP 地址信息、捕获详细的系统和网络数据，并将所有内容以结构化 Markdown 格式记录到管理的日志目录的过程。该技能处理：

- 使用用户配置文件打开浏览器访问增强的 IP 信息服务
- 捕获全面的 IP、位置和网络提供商数据
- 记录详细信息，包括 IP 地址、时间戳、浏览器信息、系统信息、ISP 详情和地理位置
- 每次创建**新的唯一 Markdown 日志文件**，并自动创建完整的目录结构
- 管理日志文件保留（仅保留最新的 10 个文件）
- 完成后清理浏览器资源

## 使用方法
```
sessions_spawn(
  runtime="subagent", 
  agentId="ip-info-recorder",
  task="记录当前公网 IP 地址，附带屏幕截图和时间戳"
)
```

## 配置
- **日志目录**：`C:\Users\Administrator\Downloads\openclaw\logs\ip-info-recorder`
- **文件保留**：仅保留最新的 10 个日志文件
- **IP 服务**：使用 https://ipinfo.io/json 获取全面的 IP 和位置数据
- **浏览器配置文件**：使用 "user" 配置文件进行已认证会话
- **日志文件命名**：**始终创建新的日志文件**，格式为 `ip_log_YYYYMMDD_HHMMSS.md`
- **日志格式**：包含详细结构化信息的 Markdown (.md) 格式

## 工作流程步骤
1. **打开浏览器**：使用用户配置文件启动浏览器
2. **导航到 IP 服务**：访问 https://ipinfo.io/ip
3. **捕获屏幕截图**：对 IP 显示进行视觉快照
4. **提取 IP 数据**：从网页内容中解析 IP 地址
5. **创建新的日志条目**：**始终创建新的日志文件**，使用唯一的时间戳名称：`ip_log_YYYYMMDD_HHMMSS.txt`
6. **管理日志文件**：删除超过 10 个文件限制的最旧文件

## 所需权限
此技能需要以下权限：
- **浏览器自动化**：使用 `profile="user"` 进行完全浏览器控制
- **文件系统访问**：写入权限到下载目录以创建日志
- **文件管理**：用于日志轮转的删除权限
- **命令执行**：用于文件操作的 PowerShell/cmd 执行

## 输出格式
**日志文件始终使用唯一的时间戳格式命名**：`ip_log_YYYYMMDD_HHMMSS.md`
**内容格式**：包含以下内容的综合 Markdown 文档：
- IP 地址和完整时间戳
- 浏览器和系统信息
- 网络服务提供商详情（如果可用）
- 地理位置信息（如果可用）
- 执行状态和工作流程步骤详情
- 所有相关元数据

每次执行都会创建一个全新的日志文件 - **永远不会追加到现有文件**。

## 示例用例
- "记录我当前的公网 IP 地址用于网络监控"
- "记录 IP 变化并附带时间戳用于安全审计"
- "捕获 IP 信息并附带视觉证据用于文档"
- "通过管理的日志保留自动跟踪 IP 地址"

## 错误处理
- 如果初始尝试失败，会重试浏览器导航
- 在记录前验证 IP 地址格式
- 优雅处理文件权限错误
- 即使日志记录失败也确保浏览器清理
